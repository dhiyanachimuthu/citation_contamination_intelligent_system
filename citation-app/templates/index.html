Analyze a Paper
How it works
  1. Enter a paper DOI
  2. Retraction Watch dataset is checked for retraction status
  3. OpenCitations API fetches real citing papers (up to 3 hops, max 150 nodes)
  4. Semantic Scholar enriches each node with title, abstract, citation count
  5. Risk scores are computed and papers ranked by contamination influence

Zero Hallucination Policy

Every node and edge in this system originates from a real API response. No synthetic papers, fake DOIs, or placeholder nodes are ever generated. Missing metadata fields are shown as NULL — never approximated or guessed. API failures return empty results with exponential-backoff retry, not fallback data.
